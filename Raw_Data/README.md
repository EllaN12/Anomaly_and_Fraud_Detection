# Raw Data

The dataset is **not stored in git** (143 MB, over GitHub's 100 MB file limit).

## Download

1. Go to [Kaggle — Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
2. Download `creditcard.csv`
3. Place it in this folder:

```
Raw_Data/creditcard.csv
```

The pipeline scripts read from that path automatically.
